# SONARQUBE

## Prerequesites

1. Install docker and docker-compose (see docker README)

## Install

1. Copy the provided docker-compose file
1. Start all containers

   `docker-compose up -d`

1. Open the web application (available on port 9000)

   User: admin

   Password: admin
